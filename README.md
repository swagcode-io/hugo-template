# 一、如何使用本仓库
## 第一步：fork it
## 第二步：安装相关环境
### Mac
``` brew install hugo ```
### Windows
请访问官方网站获取 hugo 的 .exe 安装文件
> 官方网站：https://gohugo.io/
## 第三步：在本地运行 hugo
```hugo server```
